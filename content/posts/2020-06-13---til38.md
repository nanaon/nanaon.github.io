### 스타벅스 모델링
```s
# menu 테이블과 Foreign key로 연결한 comments 테이블에 데이터 입력하기(댓글 입력)
Category(menu=Menu.objects.filter(id=1)[0], name='콜드브루').save()
# 결과값 따로 뜨지 않음

# 또는
Category.objects.create(menu=Menu.objects.filter(id=1)[0], name='콜드브루')
# <Category: Category object (2)>

# 특정 id값(PK)을 가진 객체를 쿼리셋 형태로 불러오기
Menu.objects.filter(id=1)
# <QuerySet [<Menu: Menu object (1)>]>

# 그것을 객체 형태로 발라내기
Menu.objects.filter(id=1)[0]
# <Menu: Menu object (1)>
```
