### SELECT
데이터를 읽어들일 때 사용

```sql
SELECT 
    필드명1,
    필드명2,
    필드명3,
    필드명4
FROM 테이블명
```

`WHERE`구문도 사용 가능

```sql
SELECT 
    필드명1,
    필드명2,
    필드명3,
    필드명4
FROM 테이블명
WHERE 특정필드명 = "특정필드값"
```
